## Design Specifications

### Overall Style
- **Modern Minimal Aesthetic**: Inspired by Nothing OS and Discord
- **Color Scheme**: Dark theme with cyber-style accents
  - Background: Deep black/dark gray (#0a0a0a to #1a1a1a)
  - Primary accent: Electric blue or neon cyan (#00d9ff)
  - Secondary accents: White/light gray for text (#e0e0e0)
  - Highlights: Subtle glows and light effects

### Main Components

#### 1. Header Section
- App title: "AI Audio Master" with sharp, geometric typography
- AI Mode toggle button with clean icon
- Minimal spacing, clean lines

#### 2. Central Volume Dial (Primary Control)
- **Rotatable Circular Dial**
  - Large, prominent placement in center
  - Clean circular outline with tick marks
  - Rotating indicator/pointer
  - **Light-up effect**: Glowing ring that illuminates on interaction
  - Numerical value display in center
  - Cyber-style design with geometric patterns
  - Range: 0-100

#### 3. Vertical Slider Controls (Studio Mixer Style)
- **Two vertical sliders** positioned side-by-side
  - Bass control (left)
  - Treble control (right)
- **Studio Mixing Board Aesthetic**:
  - Vertical track/rail design
  - Physical-looking slider handles/faders
  - Can be pushed up (+12) or pulled down (-12)
  - Linear gradient fills showing current level
  - Smooth, tactile appearance
  - Value indicators below each slider

#### 4. Typography
- **Sharp, Modern Font**: Inter, SF Pro, or similar geometric sans-serif
- Clean labels for each control
- Numerical values clearly displayed

#### 5. Interactive Elements
- All controls should appear touchable/draggable
- Subtle shadows and depth for dimension
- Hover states suggested through highlights
- Light-up effects on interaction zones
